# 01-06-02-if-else-elagazas-2
Kétágú elágazások  
1. feladat (Piros karikás)  
Készíts programot, mely bekéri a felhasználó életkorát, majd kiírja, hogy a felhasználó megnézheti-e a „piros karikás” (18 év alattiaknak nem ajánlott) filmet vagy nem!  
2. feladat (Szerkeszthető-e a háromszög)  
A felhasználó adja meg, a sokszög három oldalának hosszát. A program ez alapján döntse el és írja ki, hogy szerkeszthető-e vagy nem a megadott oldalakkal háromszög!  
*„Háromszög szerkeszthető, ha bármely két oldalának összege nagyobb vagy egyenlő a harmadik oldalnál!”*
